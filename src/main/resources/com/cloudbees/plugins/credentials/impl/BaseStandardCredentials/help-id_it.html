Un ID interno unico in base al quale i processi e le altre opzioni di configurazione identificano queste credenziali. Normalmente questo campo viene lasciato vuoto; in tal caso l'ID verrà generato automaticamente, il che è una buona scelta per i processi creati utilizzando l'interfaccia grafica. È utile specificarlo esplicitamente se si utilizzano le credenziali da una configurazione scriptata.
