Un elenco separato da virgole di nomi host da escludere. (Il carattere jolly * è consentito nei nomi host, ad esempio: jenkins-ci.org:*, *.jenkins-ci.org:80, jenkinsci.github.io:443.)
Se l'elenco è vuoto, nessuna combinazione nome host:porta sarà esclusa. L'elenco delle esclusioni è verificato dopo l'elenco delle inclusioni.
